# 如何将图片转为VISIO

1. 将图片上传至AI中，输入命令：帮我把这张图所有元素生成SVD代码
2. 将得到的代码复制到 [SVG代码编辑器](https://tools.jqnest.top/tools/svg_code_editor/index.html)，点击 __下载SVG__
3. 打开[SVG在线编辑器](https://www.jyshare.com/more/svgeditor/)，点击 __文件__,点击 __打开SVG__ ,选择刚刚下载的SVG文件
4. ctrl s 保存SVG文件
5. 打开VISIO，选择 __插入__——> __图片__，选择刚刚保存的SVG文件。右键选择取消组合，就可以编辑了。
